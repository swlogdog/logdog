강력한 로그 기능

XML 파일을 통하여 사용자가 직접 코딩을 하지 않고 다양한 로그옵션을 조정 할수 있습니다. 또한 레벨별로 로그를 표시하여 레벨만 교체 하여 필요한 로그를 볼 수 있게 제작 되었습니다.

Visualization 기능

구글 앱 엔진을 통하여 사용자는 단순히 앱엔진 상에 코드만 올려도 버그 리포트 데이터를 가시화 하여 볼 수 잇습니다.

버그 리포트 기능

클라이언트에서 에러가 날때 에러에 대한 정보를 모아 파일 또는 서버로 전송 할 수 있습니다.

Sending Bugreport!

클라이언트에서 에러가 날때 서버로 버그 리포트를 전송합니다. 웹에서 정보를 받아서 처리 할수도 있고 사용자가 직접 서버에 연결 할 수도 있습니다. LogDog만의 강력한 에러 리포트 기능을 느껴 보세요

Powerful Log!

웹 서버에서 실시간으로 사용자가 보내온 로그 정보를 받아 볼 수 있습니다. 클라이언트 소스와 함께 제공되는 AppEngine을 이용 할수도 있고 사용자가 클라이언트에서 직접 구현하여 다른 서버에서도 로그 정보를 받아 올 수 있게 컨트롤 할 수 있습니다.
또한 XML파일로 설정값을 바꿀수 있어 사용자가 편리하게 사용 할 수 있습니다.

Visualization through the Web!

LogDog에서 제공하는 AppEngine을 이용하면 다음과 같이 에러 리포트정보를 가시화 시켜서 한눈에 어떤 에러와 어떤 정보들이 많이 들어 왔는지 쉽게 알 수 있습니다. LogDog의 강력한 Visualization기능을 맛보세요
